Investigaciones Programación Web UTN -2

1. Usabilidad Web

Temas a tratar:

  • Importancia

  • Principios

  • Ejemplo por cada principio

2. Accesibilidad Web

Temas a tratar:

  • ¿Qué es?

  • Importancia

  • Principios

  • Ejemplo por cada principio

3. Diseño web centrado en el usuario(DCU)

Temas a tratar:

  • ¿Qué es?

  • Etapas o pasos para el DCU

  • Beneficios del DCU

  • Ejemplo o casos
